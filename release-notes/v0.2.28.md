# 0.2.28 (2026-04-26)

這個版本修正 scroll mode 重開後閱讀位置每次往下偏移的問題。主要調整是把可見進度 anchor 改成 viewport 頂端，避免 15% anchor 在還原與再次保存之間累積位移，同時保留退出時的進度保存保底。

## 修正

- scroll mode 可見進度 anchor 從 15% viewport 改為 top anchor，減少重開後位置漂移
- 退出閱讀器時會保存目前可見 scroll 位置，降低 debounce 尚未寫入時遺失進度的機率
- 一般 `flushNow()` 不會把未確認的 placeholder scroll 位置寫成正式進度，避免覆蓋已確認閱讀位置
- 更新 scroll anchor 相關測試，使 viewport bridge 與 page anchor 量測符合 top-anchor 語義
