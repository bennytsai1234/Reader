# 0.2.37 (2026-04-28)

這個版本補正閱讀器分頁與顯示層修正的正式 release tag，內容與上一輪閱讀器修正相同，但 tag 會正確指向包含修正的 commit。

## 閱讀器

- 修正 `LayoutEngine` 分頁切頁方式，改為逐行裝頁，避免頁尾文字錯分到前一頁造成頁內重疊
- 新增 page-local line box invariant 測試，驗證單頁行座標不越界且不互相重疊
- 修正 scroll 模式章節 tile 的視覺高度模型，讓 item 高度、上下 padding 與實際文字繪製區域一致
- 修正 scroll 模式底部長駐資訊列的顯示語義，不再用 page fraction 當主要指標，改用章節序號與 `ReaderLocation` 百分比
- 增加底部長駐資訊列保留高度，降低頁碼與正文互相壓住的情況
- 修正 `ReaderRuntime` 在 `jumpToLocation()` 與 `switchMode()` 開始時清除舊的 `pageWindow/currentSlidePage`，減少 slide/scroll 切換時的殘留畫面狀態
