# 0.2.7 (2026-04-20)

這是一個小型 patch release，主要是把 `v0.2.6` 之後補上的 analyzer lint 修正正式發版，讓遠端最新 release 與目前 `main` 的狀態一致。

## 內容

- 修正 `ReadAloudController` 中缺少大括號的 flow-control lint
- 修正 `test_helper.dart` 中可改為 `const` 的宣告 lint

## 驗證

- `flutter analyze lib/features/reader/runtime/read_aloud_controller.dart test/test_helper.dart`
