# 0.2.24 (2026-04-25)

這個版本集中修正閱讀頁首次顯示、滾動模式位置保存與 TTS 朗讀流暢度，並加入真實本地書、固定網路書源與 widget 級滾動還原測試，降低回到章節開頭或空白 viewport 的回歸風險。

## 亮點

- 滾動模式退出時會立即保存目前可見章節內 offset，重開書籍可回到章節中段
- 首次進入閱讀頁時，資料可見後會主動補刷新，避免內容已載入但畫面空白
- 閱讀內容上方 safe area 與底部資訊列預留空間縮小，增加正文可讀高度
- TTS 朗讀單位改為段落，聲音更連貫；高亮仍用 offset map 精準對回章節字元位置
- 新增固定本地 TXT fixture 與固定網路書源 live E2E 測試
- 新增真 UI widget 級 scroll round-trip 測試，實際拖動 `ScrollablePositionedList` 後驗證退出重開還原

## 詳細更新

### 閱讀定位與首次顯示

- scroll mode 離開閱讀頁時，會把 `visibleChapterLocalOffset` 轉成 `charOffset` 並同步持久化
- scroll fallback anchor 也允許保存，避免 `pageAnchor` 缺失時完全不記章節內位置
- 章節頁面呈現後會刷新 runtime chapter cache，降低 cache 與顯示層不同步造成的空白
- 閱讀頁第一次取得 visible data 後會排 post-frame repaint，補上首次缺少重繪的情況

### Layout 與 TTS

- 壓縮閱讀內容頂部 safe area factor 與 top spacing
- 壓縮底部 permanent info 的預留高度、上 padding 與下 spacing
- 新增段落版 `ReadAloudSegment` 建構流程
- 新增 `ReadAloudOffsetMap`，讓段落朗讀時仍能把 TTS progress 對回正確章節 offset

### 測試

- 新增固定本地測試書 `test/fixtures/reader/local_real_e2e_novel.txt`
- 新增 `reader_real_book_e2e_test.dart`
- 本地 E2E：匯入固定 TXT，滾到第十章第 10.5 行附近，退出再進入驗證還原
- 網路 E2E：固定使用 `随心看吧` 書源抓取真實書籍並驗證同樣的 scroll round-trip
- Widget E2E：pump `ReaderPage`，實際拖動 scroll view，再重建 widget 驗證 restore
