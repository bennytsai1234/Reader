## ✨ v0.2.3 更新內容

### 新功能 & 改進
- **AppBottomSheet 統一元件**：所有底部選單（書源管理、閱讀器設定、更多操作）統一使用新的 `AppBottomSheet`，支援超長內容捲動，視覺更一致
- **AppTheme 雙主題強化**：Light / Dark 主題補上完整 Material3 Token（AppBar、Card、Dialog、BottomSheet、Divider、Input），減少各頁面 hardcode 顏色
- **閱讀主題擴充**：從 3 個擴展至 7 個——簡約白、羊皮紙、嫩草綠、雅緻褐、深海藍、夜間、極黑
- **首行縮排設定回歸**：介面設定中補回首行縮排下拉選單

### 重構 & 清理
- **移除音頻播放模組**：刪除 `audio_player_page.dart`、`audio_play_service.dart` 及相關 Widget，移除 `just_audio` / `audio_session` 依賴
- **保留 TTS 朗讀**：`flutter_tts` 與 `audio_service` 通知欄整合繼續維持

### APK 改善 🎉
- **拆分 ABI 發布**：從本版起改為三個獨立 APK，依裝置架構選擇下載：

| APK 檔案 | 適用裝置 | 大小 |
|----------|---------|------|
| `app-arm64-v8a-release.apk` | 現代 Android 手機（2017 年後） | ~22 MB |
| `app-armeabi-v7a-release.apk` | 舊裝置 | ~18 MB |
| `app-x86_64-release.apk` | 模擬器 | ~20 MB |

> 不確定架構？一般手機選 **arm64-v8a** 即可。

### Bug 修復
- 修正 `reader_page.dart` 遺漏 `flutter/material.dart` import 導致 build 失敗
- 修正 `bookshelf_page.dart` 括號結構錯誤導致 `_showDeleteConfirm` 被置於 class 外
- 修正 `app_theme.dart` `CardTheme`/`DialogTheme` 型別不符（Flutter 新版 API）
- 修正 `source_manager_page.dart` 遺漏 `AppBottomSheet` import
- 修正 `audio_handler.dart` 無效 `@override` 標注
