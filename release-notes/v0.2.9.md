# 0.2.9 (2026-04-20)

這個版本主要收兩件事：第一，把閱讀器、書架、TTS、搜尋與章節記憶這一輪最直接影響體驗的問題修掉；第二，清掉一批看起來能按、實際沒有作用的假入口，避免 UI 誤導。

## 亮點

- scroll 閱讀模式更連續，章節記憶與退出後回到原位置更穩
- 加入書架即時生效，封面尺寸與操作按鈕更一致
- TTS、搜尋卡頓、來源校驗與閱讀退出流程都有實際補強
- 移除多個沒有實作的假按鈕與佔位入口

## 詳細更新

### 閱讀器

- 修正 scroll 分頁銜接與可見位置換算，改善重開書本後的章節還原
- 退出閱讀時，若這本書尚未加入書架但已有閱讀進度，會先詢問是否加入書架
- 打點區九宮格預設改為全部喚起選單，設定頁也補齊真實保存
- 移除舊的假亮度遮罩做法，避免與系統通知欄亮度體驗不一致

### 書架 / 搜尋 / 詳情

- 加入書架後書架頁會立即刷新，不再需要重啟 App
- 書架封面比例與書名高度固定，減少尺寸跳動
- 搜尋結果進入閱讀前會停止背景搜尋，避免持續搜尋拖慢 UI
- 書籍詳情頁移除未實作的「預加載」選單

### TTS

- 修正 TTS 初始顯示就變成「已暫停」的狀態錯誤
- 修正播放按鈕無法點擊與待朗讀文本未正確保存的問題
- 移除朗讀設定裡未接系統頁面的假入口

### 書源管理 / 校驗

- 書源管理多選、全選與列表互動再對齊 Legado
- 校驗流程新增 discovery detail/toc/content 細分類
- 校驗結果面板支援依失效類型篩選與批次選取處理
- 列表會即時顯示校驗中狀態、成功/失敗標記與完整日誌

### UI 清理

- 移除書架選單中沒有實際行為的「更新目錄」與「日誌」
- 移除其他設定中未接後端的「記錄運行日誌」與「並行下載線程數」
- 移除閱讀器圖片預覽裡假的「保存」按鈕
