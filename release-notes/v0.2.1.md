# 0.2.1 (2026-04-10)

## 🌟 增強功能 (Enhancements)

* **書源管理體驗大幅升級 (對標 Legado)**:
  * 移除了冗餘的「多選模式」切換開關，現在書源列表採用預設顯示核取方塊 (Checkbox) 的設計，可以直接進行多選操作，更符合原版 Legado 習慣。
  * 列表支援顯示更多標籤，包含分組資訊、書源 URL 與回應狀態標籤，讓排版更為緊湊俐落。
  * 將繁雜的操作按鈕（啟用、禁用、加入分組、匯出等）整合到整潔的溢出選單 (Overflow Menu /  ⋮) 內，讓工具列不再擁擠。
  * 調整了拖曳排序 (Drag handles) 的觸發區域與視覺，使其在手動排序模式下體驗更順暢。
