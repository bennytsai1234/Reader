# 0.2.38 (2026-04-28)

這個版本完成 reader 新 baseline 的十階段重構，重點是把閱讀進度統一成穩定的三元位置，並重整 scroll / slide viewport、restore、overlay 與控制接口。

## 閱讀器

- 將 durable progress 統一為 `ReaderLocation(chapterIndex, charOffset, visualOffsetPx)`，並把 `visualOffsetPx` 納入 DB、storage、serialization、copy/equality/hash 行為
- 正規化 `visualOffsetPx` 範圍為 `-80..120`，非有限值會回退為 `0`
- 將 scroll 模式改為以章節 tile window 與座標 mapper 驅動，支援上下跨章與 signed virtual scroll
- 新增 page cache render model，將正文 tile 與 overlay 分層，避免 TTS 高亮更新重建正文 tile
- 建立 visible location / committed location / saveProgress 的責任邊界，滑動中、restore、loading/error/placeholder 與 layout invalidation 不寫 DB
- 實作 restore from location，支援 scroll / slide 模式依 `ReaderLocation` 回到接近的視覺位置
- 重整 slide page cache placement，改由 durable location 與 page cache window 做位置對應，不保存 page index 或 virtual scroll 狀態
- 新增 reader overlay gesture layer 與 viewport controller 接口，讓 auto page 與 TTS follow 透過 viewport 控制層移動

## 文件

- 補齊 reader recovery、visible location、restore、layout coordinate、page cache render、scroll canvas window、overlay gesture 與 implementation order 設計文件
- 明確記錄目前 reader 為全新 app baseline，不把舊開發 DB migration 視為 recovery blocker
