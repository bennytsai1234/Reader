# 0.2.39 (2026-04-29)

這個版本補上 reader 重構後的閱讀畫面穩定性修復，重點是修正 slide/scroll viewport、底部常駐資訊列與長文字排版在實機閱讀時暴露出的問題。

## 閱讀器

- 修正長中文句、URL、長英文與長數字 token 橫向放不下時的排版行為，會重新量測並續排到下一行，不會把整段文字重畫在同一行
- 避免來源硬換行進入同一個 `TextLine`，並限制文字 painter 單一 line box 只畫一行，降低重疊文字風險
- 修正 slide 模式 page/tile 高度沒有明確約束時可能出現落頁或初始進入空白
- 底部常駐資訊列改為獨立預留區域，loading 到 ready 期間維持一致 content viewport，避免初始 layout 高度跳動
- 修正 scroll 模式底部資訊顯示章節內進度，而不是整本書進度
- 修正 scroll 模式短章與章節邊界附近拖曳時容易被 edge shift 干擾的問題
- 補上 `ReadStyle` value equality，避免 parent rebuild 讓 scroll viewport 誤判 style 改變並重置 scroll 狀態
- 正規化過小行高，並讓分頁避開底部裁切區，避免頁尾文字被切半
