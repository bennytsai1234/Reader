# 0.2.41 (2026-04-29)

- 修正 reader restore、relayout、reload 與 mode switch 交錯時，舊的非同步結果覆蓋新閱讀狀態的問題。
- 強化 PageResolver 對 in-flight layout task 的失效處理，避免已淘汰的排版結果回填 cache。
- 修正 scroll window 位移與 slide placeholder 滑動門檻，避免滑到 loading/error placeholder 時落到錯誤頁面。
- 強化閱讀進度 flush 與離開頁面保存，沒有 viewport capture 時也能保存最新可見位置。
- 修正 slide 模式 TTS 高亮跟隨與翻頁保存，讓朗讀位置能正確推進到目標頁。
