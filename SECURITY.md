## 回報安全漏洞

**請勿透過公開 GitHub Issues 回報安全性漏洞。**

若你發現安全漏洞，請以電子郵件回報：

**bennytsai0711@gmail.com**

回報時請盡量提供以下資訊，以便我們更快速地評估影響範圍：

- 漏洞類型（如：XSS、SQL Injection、不安全的資料儲存……）
- 受影響的檔案路徑（完整路徑或相對路徑）
- 觸發漏洞所需的特殊設定
- 複現步驟（越詳細越好）
- 概念驗證（Proof-of-concept）或 exploit 程式碼（如有）
- 漏洞的潛在影響，包括攻擊者可能如何利用

我們將在 **7 個工作日內**回覆並告知預期的處理時程。

## 說明

墨頁 Inkpage 是一款本機閱讀器應用程式，主要使用者資料（書源規則、書籍內容、閱讀進度）儲存於裝置本機的 SQLite 資料庫。

需注意的安全考量：

- **書源規則包含任意 JavaScript 執行**（透過 `flutter_js` 沙盒），書源來自第三方，使用者自行負責信任風險
- **網路請求**由書源規則驅動，應用程式本身不收集使用者資料至遠端伺服器
- **備份檔案**為本機 ZIP，不包含雲端同步功能
