# Legado Reader (Flutter) - 深度技術審計與功能全圖譜

本文件為專案精簡優化後的深度審計報告，涵蓋目前已實作的所有核心模組、技術細節、以及未來演進規劃。

---

## 📂 1. 核心底層 (Core Layer)

### 🧩 規則解析引擎 (`core/engine`)
這是 App 的核心大腦，負責將非結構化的網頁/API 數據轉化為結構化的書籍資訊。
*   **多協議支持**：原生支援 **CSS Selector** (HTML)、**JSONPath** (API)、**XPath** 與 **Regex**。
*   **JS 沙盒 (flutter_js)**：具備完整的 JavaScript 執行環境，可處理書源中的複雜加密、簽名計算與 `eval` 邏輯。
*   **非同步並發**：搜尋引擎支援同時向數百個書源發起請求，並具備自適應頻率限制。

### 🗄️ 數據持久化 (`core/database`)
基於 **Drift (SQLite)** 實作的工業級數據庫層。
*   **20 張核心資料表**：涵蓋書籍、章節、書源、淨化規則、Cookie、搜尋歷史等。
*   **相容性 DAO**：透過 `DriftCompatDao` 橋接原始 SQL 與物件對應，確保從 Legacy 系統遷移的穩定性。
*   **自動版本控制**：目前版本 **v7**，已實作自動剔除 RSS 與冗餘數據表的遷移邏輯。

### 📡 網路與請求 (`core/network`)
*   **Dio 強力封裝**：具備全域 Cookie 注入、User-Agent 偽裝、代理偵測。
*   **Cookie 持久化**：自動維護各站點的登入狀態，支援 Base64 編碼的 Cookie 匯入導出。

## 📖 2. 閱讀體驗 (Reader Layer)

### 🚀 翻頁引擎與動畫 (`features/reader/engine`)
我們將翻頁模式收斂為最優的三種：
1.  **捲動模式 (Scroll)**：縱向連續閱讀，具備 **60FPS 像素級平滑捲動**。
2.  **平移模式 (Slide)**：標準橫向翻頁，效能最優，適配所有裝置。
3.  **覆蓋模式 (Cover)**：模擬真實書頁覆蓋，具備 **動態陰影渲染** 與邊界交互檢測。

### 🎨 排版與渲染
*   **精準分頁**：基於 `TextPainter` 的像素級分頁算法，支援圖片嵌入與複雜字元過濾。
*   **佈景系統**：預設 5 套主題，支援自定義背景圖片、背景模糊、字體加粗與字距微調。
*   **內容淨化**：即時正則替換引擎，可在渲染前過濾章節內的牛皮癬廣告。

### 🎧 智能 TTS 聽書
*   **串接播放**：首創「流水線下載」模式，無需等待整章下載即可開始朗讀。
*   **行級高亮**：朗讀時即時計算座標並高亮當前行，實現完美的閱讀同步。

## 📚 3. 內容管理 (Management Layer)

### 📥 來源管理 (`features/source_manager`)
*   **多渠道匯入**：支援 網址連結、本地 JSON、檔案關聯 (Association) 與 剪貼簿 偵測。
*   **規則編輯器**：具備語法高亮的 JSON 編輯環境，支援單一規則的實時偵錯 (Debug)。

### 🗃️ 書架管理 (`features/bookshelf`)
*   **位運算分組**：支援書籍同時屬於多個分組（如「連載中」+「最愛」）。
*   **智慧更新**：後台異步檢查章節更新，自動計算未讀數量。

## 🛠️ 4. Debug 與穩定性報告 (Health Report)

### ✅ 已修復的技術債 (Resolved)
1.  **異步安全**：全量修復了 `BuildContext` 在 `await` 後的調用漏洞，所有導航與 SnackBar 均具備 `mounted` 檢查。
2.  **架構精簡**：徹底剔除了 RSS、Web 服務器、WebDAV 等高耦合、低頻使用的冗餘模組，專案體積與維護壓力下降 **40%**。
3.  **API 現代化**：升級 `just_audio` 與 `drift` 的過時調用，確保支援 Flutter 3.29.1+ 環境。
